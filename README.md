## 📦 Installation

```bash
git clone https://github.com/hatem32/GymManagementSystem.git
```

Update your connection string inside:

```
appsettings.Development.json
```

Run:

```bash
Update-Database
```

or

```bash
dotnet ef database update
```

Run the project.
